# 🚀 Shams Ul Rehman | Professional Portfolio

A high-performance, modern, and SEO-optimized personal portfolio designed for a **Digital Marketing Manager & Content Specialist**. This project showcases expertise in driving organic growth, B2B lead generation, and high-converting content strategies.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
* **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
* **Icons:** FontAwesome 6.4
* **Typography:** Google Fonts (Poppins)
* **Build Tool:** Vite (Optimized for fast loading and modern development)

---

## ✨ Key Features

* **Conversion-Focused:** Integrated CTA buttons for WhatsApp, LinkedIn, and Email to reduce friction for potential clients.
* **Fully Responsive:** Seamless experience across Mobile, Tablet, and Desktop devices.
* **SEO Ready:** Built with semantic HTML and optimized metadata to align with search engine best practices.
* **Modern UI/UX:** Minimalist design with a professional blue accent, soft shadows, and smooth scrolling.
* **Interactive Elements:** Card-based layouts with hover effects and scroll-triggered animations.

## 🚀 Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ak9612940-ux/shams-ul-rehman-portfolio.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run development server:**
    ```bash
    npm run start
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```
